== Description ==

Google Sitemap video est n plugin qui va vous permette d'indexer des pages comprenant des vidéos dailymotion dans google vidéo. Il vous suffit de mettre une petite vidéo dailymottion en accompagnement de celle de youtube par exemple.

== Installation ==

Pour l'installation du plugin, suivez le lien http://codex.wordpress.org/fr:Gestion_de_Plugins

=== Inspiration ===

Le plugin est basé et inspiré du plugin http://wordpress.org/extend/plugins/xml-sitemaps-for-videos/

=== Faq ===

Pourquoi des vidéos Dailymotion ?

C'est très simple les vidéos youtube ne vont pas s'indexer en tant que contenu vidéo sur votre site alors que les vidéos dailymoiton google vidéos les interprétent comme vos propres vidéos.
